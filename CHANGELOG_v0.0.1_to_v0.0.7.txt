VERSION 0.0.7 - MAJOR SECURITY & UI OVERHAUL
==============================================

🛡️ SECURITY ENHANCEMENTS:
-------------------------
• Re-enabled Chromium sandbox for enhanced security
• Implemented HTTPS enforcement with automatic HTTP to HTTPS conversion
• Added certificate validation and error handling
• Enhanced Content Security Policy (CSP) with stricter rules:
  - Added object-src 'none' to block plugins
  - Added base-uri 'self' to restrict base URI
  - Added form-action restrictions
  - Added frame-ancestors 'none' to prevent clickjacking
  - Added upgrade-insecure-requests directive
• Implemented strict permission controls:
  - Only allows notifications by default
  - Denies camera, microphone, geolocation permissions
• Enhanced download security:
  - Blocks dangerous file extensions (.exe, .bat, .cmd, etc.)
  - Validates download URL protocols
  - Enhanced security logging
• Applied secure web preferences to all windows and BrowserViews:
  - webSecurity: true
  - allowRunningInsecureContent: false
  - experimentalFeatures: false
  - enableRemoteModule: false
  - sandbox: true
  - safeDialogs: true

🎨 USER INTERFACE IMPROVEMENTS:
-------------------------------
• Modern card-based design system for settings pages
• Implemented responsive CSS grid layouts with hover effects
• Added danger button styling (red buttons for destructive actions)
• Enhanced theme system with 9 different theme variants:

  - Dark themes: Default, Purple, Nord, Forest, Rose
• Integrated settings into tab system instead of separate windows
• Hidden URL bar for settings tabs for cleaner interface
• Streamlined settings sidebar with removed unnecessary sections
